# 🎯 Customer Churn Prediction & Retention Dashboard

End-to-end Machine Learning pipeline to predict customer attrition for a telecom company, with actionable business insights deployed via an interactive Streamlit dashboard.

## 🚧 Status
🔨 Work in progress — Building EDA phase

## 📊 Dataset
Telco Customer Churn — 7,043 customers, 21 features.  
Source: [Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

## 🛠️ Tech Stack
Python · pandas · scikit-learn · XGBoost · Streamlit
